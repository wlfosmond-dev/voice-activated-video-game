# Voice Activated Video Game

A Python and Arduino-based interactive game that uses real-time microphone input to control character movement.

The project combines hardware and software by integrating Arduino sensor input with a Python Tkinter graphical interface.

---

## Features

- Real-time voice-controlled jumping
- Arduino microphone input detection
- Dynamic obstacle spawning
- Collision detection system
- Score tracking
- Buzzer sound feedback
- Tkinter graphical game interface

---

## Technologies Used

- Python
- Arduino
- Tkinter GUI
- Analog sensor input
- Real-time game loop logic

---

## How It Works

The Arduino continuously reads microphone sound levels through an analog input.

When the detected sound exceeds a predefined threshold, the Python game triggers a jump action for the player character.

Obstacles move across the screen while the player attempts to avoid collisions using voice input.
